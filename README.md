# Pokelator
Pokemon damage simulator

👷 This project is **UNDER CONSTRUCTION** 🏗

### Setup

```bash
$ cp .env.example .env
$ touch database/database.sqlite
$ composer install
$ npm install
$ php artisan migrate
$ php artisan make:pokemon
$ php artisan make:types
```

### Start up local server

```bash
$ php artisan serve
$ npm run hot
```
